# Getting started in ruby

This example assumes that you have ruby installed on your system along with the bundler gem. If this is not the case you can find more detailed information on installing ruby on the [Ruby](https://www.ruby-lang.org/en/documentation/installation/) site.

1. Follow the first step on https://developers.google.com/genomics/ to setup
 some client secrets. Copy the client_secrets.json file into this directory.

2. Install the required gems using bundler

```
bundle install
```

3. Run the test script

```
ruby main.rb
```


# More information

This example is using
[Google's ruby client library](https://developers.google.com/api-client-library/ruby/)
